lang/en Taalbestand

De lang/en map bevat de taalbestanden voor de plugin. Deze bestanden maken het mogelijk om tekst in de gebruikersinterface van de plugin aan te passen en weer te geven in verschillende talen. Voor Engelse tekst gebruik je het bestand en/strings.php in deze map.

Bestand: local_ldap_connection.php

In lang/en/local_ldap_connection.php definieer je alle Engelse taalstrings die door de plugin worden gebruikt. Elke string krijgt een unieke identifier die wordt gebruikt om de tekst in je code op te halen.

Structuur van een Taalstring


// Taalstrings voor de LDAP Connection plugin
$string['pluginname'] = 'LDAP Connection';
$string['suspend_inactive_users'] = 'LDAP Cron Task';
$string['cron_task_success'] = 'Aanpassen van de gebruikers was succesvol';
$string['cron_task_error'] = 'Fout bij het bijwerken van gebruikers';

Beschrijving van de Taalstrings

  • $string['pluginname']: De naam van de plugin zoals weergegeven in Moodle.
  • $string['suspend_inactive_users']: De naam van de cron-taak. Dit wordt gebruikt in de get_name() methode van suspend_inactive_users.php om de taaknaam weer te geven.
  • $string['cron_task_success']: Bericht dat wordt weergegeven wanneer de cron-taak succesvol is voltooid.
  • $string['cron_task_error']: Bericht dat wordt weergegeven wanneer er een fout optreedt bij het uitvoeren van de cron-taak.

Nieuwe Strings Toevoegen

Om nieuwe tekst toe te voegen aan de plugin, voeg je nieuwe items toe aan de $string array in strings.php. Elke nieuwe string krijgt een unieke sleutel (identifier) en de bijbehorende tekst. Hier is een voorbeeld:


$string['new_feature_title'] = 'Nieuwe Functie Titel';
$string['new_feature_description'] = 'Beschrijving van de nieuwe functie.';

Vervolgens kun je deze strings ophalen in je PHP-code met get_string():


echo get_string('new_feature_title', 'local_ldap_connection');

Taalstring Gebruiken in je Plugin

Je haalt een taalstring op met de get_string() functie in Moodle. Deze functie accepteert twee parameters:

  • Identifier: De naam van de string in het strings.php bestand, zoals pluginname of cron_task_success.
  • Component: De naam van de plugin, bijvoorbeeld local_ldap_connection.

Voorbeeld van Taalstring Gebruik


public function get_name() {
    return get_string('suspend_inactive_users', 'local_ldap_connection');
}

In dit voorbeeld haalt get_name() de suspend_inactive_users string op uit het taalbestand. Deze string bevat de naam van de cron-taak zoals weergegeven in Moodle.
